Property Crawler Demo

A Python-based property listing intelligence pipeline that scrapes listing pages, performs reverse image matching, and discovers publicly available contact information.

View GitHub Repo View Sample Input CSV View Sample Output CSV

How It Works

URLs CSV → Listing Scraper → Reverse Image Search → Contact Discovery → Results CSV

Features

  • Scrapes Airbnb, VRBO, Booking.com, and generic property sites
  • Extracts title, host, location, description, and listing photos
  • Performs reverse image search to find additional sources
  • Discovers public contact details such as owner, management company, email, and phone
  • Exports structured results to CSV or JSON
